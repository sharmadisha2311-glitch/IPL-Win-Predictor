# IPL Win Predictor 🏏

## 📌 Overview

This project predicts the winning probability of an IPL match using Machine Learning.

## 🚀 Features

* Predict match outcome in real-time
* Streamlit web app UI
* Trained ML model using historical data

## 🛠️ Tech Stack

* Python
* Pandas
* NumPy
* Scikit-learn
* Streamlit
